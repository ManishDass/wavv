## Features

<center>
Modern UI
Multiple Login Options
Email Verification
Online Song Search With Suggestions <br/>
Streaming Support <br/>
Lyrics Support <br/>
No Ads <br/>
No Subscriptions <br/>
Material UI & Accent Colors & Dynamic Colors (Android 12+) <br/>

</center>
